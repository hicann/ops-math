# SearchSorted

##  产品支持情况

| 产品 | 是否支持 |
| ---- | :----:|
|Atlas A3 训练系列产品/Atlas A3 推理系列产品|√|
|Atlas A2 训练系列产品/Atlas 800I A2 推理产品/A200I A2 Box 异构组件|√|

## 功能说明

- 算子功能：在一个已排序的张量 sorted_sequence 中查找给定张量 values 应该插入的位置。返回与 values 相同大小的张量，其中每个元素表示给定值在原始张量中应该插入的位置。
