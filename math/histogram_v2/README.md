## 功能说明

- 算子功能：计算张量直方图。
- 计算公式：以min和max作为统计上下限，在min和max之间划出等宽的数量为bins的区间，统计张量x中元素在各个区间的数量。如果min和max相等，则使用张量中所有元素的最小值和最大值作为统计的上下限。小于min和大于max的元素不会被统计。
