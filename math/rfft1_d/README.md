## 功能说明

- 算子功能：对输入张量进行RFFT（傅里叶变换）计算，输出是一个包含非负频率的复数张量。
- 计算公式：
  $$
  y = W \cdot x
  $$
  其中W为DFT矩阵，其定义为$W_{jk}=e^{-ij2\pi{\tfrac{k}{N}}n}$
- 示例：
  假设x为 {1, 2, 3, 4} 则y = {10, 0, -2, 2, -2, 0} = {10 + 0j, -2 + 2j, -2 + 0j} （自定义）
