## 功能说明

- 算子功能：使用循环填充方式对输入张量进行填充操作。
- CircularPad2d：对输入张量的最后两维进行循环填充。
- CircularPad3d：对输入张量的最后三维进行循环填充。

## 约束说明

- padding值必须小于对应维度的大小。
- out的最后一维在不同类型下的大小需满足如下约束：
  - int8：(0, 98304)
  - float16/bfloat16：(0, 49152)
  - int32/float32：(0, 24576)
- 输入和输出的数据类型必须一致。
- 支持非连续的Tensor。
