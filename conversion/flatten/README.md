## 功能说明

- **算子功能**：将输入Tensor基于给定的axis扁平化为一个2D的Tensor。
- **计算公式**：
  - 若输入self的shape为(d₀, d₁, ..., dₙ)，则输出out的shape为(d₀×d₁×...×dₐₓᵢₛ₋₁, dₐₓᵢₛ×dₐₓᵢₛ₊₁×...×dₙ)
  - 若axis取值为0，则输出out的shape为(1, d₀×d₁×...×dₙ)

## 约束说明

- 输入张量的维度不能大于8。
- axis必须在有效的维度范围内。
- 输入和输出的数据类型必须一致。
