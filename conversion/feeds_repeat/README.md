## 功能说明

- 算子功能：
  
  对于输入feeds，根据输入feeds_repeat_times,将对应的feeds的第0维上的数据复制对应的次数，并将输出y的第0维padding到output_feeds_size的大小

## 约束说明

在算子实现过程中，feeds_repeat_times需搬入ub并做精度转换和累加，故feeds_repeat_times数据规模不能超过ub大小的四分之一。
