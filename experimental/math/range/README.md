# Range
## 产品支持情况

| 产品                                                         | 是否支持 |
| :----------------------------------------------------------- | :------- |
| <term>Atlas A2 训练系列产品/Atlas A2 推理系列产品</term> | √        |
## 功能说明

- 算子功能：对输入标量start,end,step 生成区间[start,end)范围内的等差数列
## 参数说明

| 参数名 | 输入/输出/属性 | 描述                                                         | 数据类型 | 数据格式 |
| :----- | :------------- | :----------------------------------------------------------- | :------- | :------- |
| start   | 输入           | start                                | float32  |   ND    |
| end   | 输入           | end                                 | float32   |   ND    |
| step   | 输入           | step                                 | float32 |   ND    |
| out    | 输出           | y | float   |      ND     |


## 约束说明
