# Sub
## 产品支持情况

| 产品                                                         | 是否支持 |
| :----------------------------------------------------------- | :------- |
| <term>Atlas A2 训练系列产品/Atlas A2 推理系列产品</term> | √        |
## 功能说明

- 算子功能：完成减法计算。
- 计算公式：
$$ y = x1 - x2 $$

## 参数说明

| 参数名 | 输入/输出/属性 | 描述                                                         | 数据类型 | 数据格式 |
| :----- | :------------- | :----------------------------------------------------------- | :------- | :------- |
| x1   | 输入           | 被减数张量。                                                 | 见下方   | ND       |
| x2  | 输入           | 减数张量。                                                   | 见下方   | ND       |
| y    | 输出           | 减法计算的结果。                                             | 见下方   | ND       |

- <term>Atlas A2 训练系列产品/Atlas A2 推理系列产品</term>：数据类型支持FLOAT、FLOAT16、INT32、INT16、BFLOAT16。

## 约束说明

无
