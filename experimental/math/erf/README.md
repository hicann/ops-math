## 功能说明

- 算子功能：返回输入Tensor中每个元素对应的误差函数的值。

- 计算公式：

$$
y =\frac{2}{\sqrt{\pi } } \int_{0}^{x} e^{-t^{2} } \mathrm{d}t
$$
