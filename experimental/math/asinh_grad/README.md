# AsinhGrad

## 产品支持情况

| 产品                                              | 是否支持 |
|:------------------------------------------------| :------: |
| <term>Ascend 950PR/Ascend 950DT</term>          |    √     |

## 功能说明

+ 算子功能：计算 asinh_grad(y, dy)，即反双曲正弦函数的梯度。

+ 计算公式如下：

$$
dx = \frac{dy}{\cosh(y)}
$$

## 参数说明

| 参数名 | 类别  | 描述                        | 数据类型                       | 数据格式 |
|:---:|:---:|:-------------------------:|:--------------------------:|:----:|
| y   | 输入  | 待进行asinh_grad计算的入参，公式中的y  | bfloat16, float16, float32 | ND   |
| dy  | 输入  | 待进行asinh_grad计算的入参，公式中的dy | bfloat16, float16, float32 | ND   |
| dx  | 输出  | 待进行asinh_grad计算的出参，公式中的dx | bfloat16, float16, float32 | ND   |
