## 功能说明

- 算子功能：对输入张量的每一个元素，按照指定的小数位，四舍五入。

- 计算公式：

  $$out_i=round(input_i,decimals)$$

  举例如下：

  Round(3.56,0) = 4.0

  Round(3.56,1) = 3.5

# 参数说明

| 参数名   | 输入/输出/属性 | 描述                                       | 数据类型                        | 数据格式 |
| -------- | -------------- | ------------------------------------------ | ------------------------------- | -------- |
| x        | 输入           | 入参，公式中的input，对该数据进行四舍五入  | bfloat16，float16，float，int32 | ND       |
| decimals | 属性           | 入参，公式中的decimals，指定四舍五入的位数 | int                             | ND       |
| y        | 输出           | 待进行round计算的出参，公式中的y           | bfloat16，float16，float，int32 | ND       |
