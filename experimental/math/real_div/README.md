# RealDiv

##  产品支持情况

| 产品 | 是否支持 |
| ---- | :----:|
|Atlas A2 训练系列产品/Atlas 800I A2 推理产品/A200I A2 Box 异构组件|√|

## 功能说明

- 算子功能：对两个输入向量进行浮点除运算，对于bool类型输入数据，输出数据是float。

- 计算公式：

$$
y=x1/x2
$$
