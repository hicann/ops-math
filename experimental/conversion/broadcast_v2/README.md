# BroadcastV2

## 产品支持情况

| 产品                                                         | 是否支持 |
| :----------------------------------------------------------- | :------- |
| <term>Atlas A2 训练系列产品/Atlas A2 推理系列产品</term> | √        |

## 功能说明

- 算子功能：将输入张量沿指定轴（axis）进行广播，使该轴的长度变为原来的 num 倍。
- 计算公式：
  无特定数学公式，为张量形状变换操作。

## 参数说明

| 参数名 | 输入/输出 | 描述                                                         | 数据类型 | 数据格式 |
| :----- | :------------- | :----------------------------------------------------------- | :------- | :------- |
| x   | 输入           | 输入张量                                                 | 见下方   | ND       |
| y    | 输出           | 输出张量                                             | 见下方   | ND       |

- <term>Atlas A2 训练系列产品/Atlas A2 推理系列产品</term>：数据类型支持FLOAT、FLOAT16。

## 约束说明

无
